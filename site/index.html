6 modelos

ISNet por defecto (rápido y muy bueno) y BiRefNet 2024 para máxima calidad. Elegí según el caso.

Alpha matting

Modo de bordes finos para pelo, plantas y mallas. Ajustá umbrales para resultados quirúrgicos.

Web + Desktop

Usalo en el navegador o como app nativa con rm-bg desktop (Electron). La misma UI, sin pestañas.

Una herramienta, muchas formas de usarla

Empezá en un comando

Necesitás Node.js y Python 3.9+ ya instalados.

Probarlo ya

npx (temporal)

npx -y remove-background-local

Lo descarga temporalmente, lo corre y abre 127.0.0.1:7860. No deja nada instalado.

Recomendado

npm (permanente + rm-bg)

npm install -g remove-background-local
rm-bg web # o: rm-bg desktop

Instala el comando rm-bg en todo el sistema. Ideal para uso regular y la app de escritorio.

Desde el código

git clone

git clone …/remove-background-local
cd remove-background-local && ./run.sh

El primer arranque crea el entorno y baja las dependencias.
